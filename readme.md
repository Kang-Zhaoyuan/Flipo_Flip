# Flipo Flip Simulation

This is a MuJoCo-based simulation project developed for **CUPT Problem 6**. It simulates the rolling and flipping behavior of a **Flipo Flop** under realistic conditions with **friction and air resistance**.

## Project Overview

- **Research Object**: Dynamics of Flipo Flop rolling and flipping under combined friction and aerodynamic drag  
- **Physics Engine**: MuJoCo  
- **Key Focus**: Realistic physical behavior including energy dissipation, rolling stability, and flipping patterns
