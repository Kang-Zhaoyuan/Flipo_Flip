# Flipo Flip 

This is a MuJoCo-based simulation project developed for **CUPT Problem 6**. It simulates the rolling and flipping behavior of a **Flipo Flop** under realistic conditions with **friction and air resistance**.（See more in [Folder Simulation](./Simulation)）

Also, this repository uses **OpenCV** and **Machine Learning** to track the motion of the **Flipo Flip** in the real world, aiming to explore the effects of its geometry and release conditions on it.（See more in [Folder Video_Analysis](./Video_Analysis)）

# 

本项目是一个基于 MuJoCo 的仿真项目，专为 **CUPT 第6题** 而开发。在**摩擦力与空气阻力**的真实条件下，模拟 **Flipo Flop** 的滚动与翻转行为。（详见 [Simulation 文件夹](./Simulation)）

此外，本仓库使用 **OpenCV** 与**机器学习**技术对现实中 **Flipo Flip** 的运动进行追踪，旨在探究其几何形状与释放条件对运动的影响。（详见 [Video_Analysis 文件夹](./Video_Analysis)）
